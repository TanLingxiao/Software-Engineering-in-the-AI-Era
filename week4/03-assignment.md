# 作业：使用 Claude Code 编程

## 📋 作业概述

本作业将让你使用 Claude Code 完成实际的编程任务，体验人机协作的开发模式。

## 🎯 学习目标

完成本作业后，你将能够：

- 熟练使用 Claude Code 进行编程
- 理解人机协作的最佳实践
- 评估 Claude Code 的能力和局限性
- 优化与 AI 的协作流程

## 📝 作业任务

### 任务 1：设置 Claude Code

1. 安装 Claude Code CLI
2. 配置 API 密钥
3. 熟悉基本命令

### 任务 2：完成编程项目

选择以下项目之一，使用 Claude Code 完成：

**选项 A：待办事项 API**

要求：
- RESTful API 设计
- 用户认证
- CRUD 操作
- 数据持久化
- API 文档

**选项 B：博客系统**

要求：
- 文章管理
- 评论功能
- 用户系统
- 搜索功能
- 标签分类

**选项 C：实时聊天应用**

要求：
- WebSocket 支持
- 房间管理
- 消息历史
- 用户在线状态
- 消息加密
