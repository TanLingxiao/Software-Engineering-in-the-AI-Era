# 作业：LLM 提示游乐场

## 📋 作业概述

本作业旨在让你实践提示工程技术，探索 LLM 的能力边界，并学习如何有效地与 AI 工具协作。

## 🎯 学习目标

完成本作业后，你将能够：

- 设计有效的提示来生成高质量的代码
- 识别并避免常见的提示工程陷阱
- 评估 LLM 输出的质量
- 迭代优化提示以获得更好的结果

## 📝 作业任务

### 任务 1：代码生成

使用 LLM 生成以下功能的代码，并评估结果：

1. **基础任务**：实现一个二分查找算法
2. **中等任务**：创建一个简单的 REST API 端点
3. **高级任务**：实现一个缓存装饰器

**要求**：
- 为每个任务设计至少 3 个不同的提示
- 记录每个提示的输出质量
- 分析哪些提示技巧最有效

### 任务 2：代码审查

让 LLM 审查以下代码：

```python
def calculate_average(numbers):
    total = 0
    for num in numbers:
        total += num
    return total / len(numbers)
```

**要求**：
- 设计提示让 LLM 识别潜在问题
- 记录 LLM 发现的问题
- 根据建议改进代码

### 任务 3：调试辅助

模拟一个调试场景：

1. 写一段有 bug 的代码
2. 使用 LLM 帮助找出问题
3. 记录调试过程
4. 评估 LLM 的调试能力

### 任务 4：提示优化

选择一个之前生成的代码，通过迭代优化提示来改进输出：

```
第 1 轮提示：[初始提示]
输出：[初始输出]
问题：[发现的问题]

第 2 轮提示：[改进后的提示]
输出：[改进后的输出]
改进：[改进之处]

...继续直到满意...
```

### 任务 5：边界探索

探索 LLM 的能力边界：

1. 尝试让 LLM 生成它声称能做但实际做不到的任务
2. 记录失败的情况
3. 分析失败的原因
4. 提出可能的解决方案
