# 第1周：LLM 与 AI 开发入门

## 📋 本周概述

欢迎来到现代软件开发者课程！本周我们将介绍大型语言模型（LLM）的基础知识，以及如何有效地使用 AI 工具进行软件开发。

## 🎯 学习目标

完成本周学习后，你将能够：

- 理解什么是 LLM 以及它的工作原理
- 掌握有效提示工程的核心技巧
- 在实际开发中应用 LLM 提升生产力

## 📚 本周主题

1. **课程介绍** - 了解课程结构和期望
2. **LLM 基础** - 深入了解大型语言模型
3. **提示工程** - 学习如何有效地与 AI 交互
