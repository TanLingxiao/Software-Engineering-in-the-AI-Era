# 现代软件开发者 (The Modern Software Developer)

> 斯坦福大学 CS146S 课程中文学习教程

## 📚 课程简介

在过去的几年里，大型语言模型（LLM）在软件开发领域引入了一种革命性的新范式。传统的软件开发生命周期正在被各个阶段的 AI 自动化所改变，这引出了一个核心问题：下一代软件工程师应该如何利用这些进步来提升10倍的生产力，并为他们的职业生涯做好准备？

本课程展示了现代 AI 工具不仅会提升开发者生产力，还会将软件工程民主化，让更广泛的受众能够参与其中。我们将展示软件开发已经从 0-1 的代码创建演变为计划、AI 生成、修改和重复的迭代工作流程。

### 🎯 课程目标

通过本课程，你将：

- 掌握现代开发工具的使用
- 理解 AI 辅助编程的原理和实践
- 学习自动化测试和部署技术
- 探索新兴的软件发展趋势
- 获得将最先进的 LLM 模型集成到复杂开发工作流程中的实践经验
